Contexte

Du 25 au 26 juin 2016, lors du hackathon organisé par Solutec

Pendant 2 jours, 4 équipes de six personnes ont imaginé des applications sur les thèmes suivants :

  • Entreprise connectée (outil de communication collaboratif et novateur).

  • Des jeux vidéo (description du CV sous forme de jeu vidéo).
  • Une idée, un projet ? Venez avec vos idées et développez votre projet pendant les 2 jours.

Next

Une équipe improvisée d'horizons divers

Le désir de concevoir une application utilisant le potentiel de la modélisation sémantique d'un côté, l'envie de réaliser un jeu vidéo basé sur la puissance d'Unity de l'autre côté, et les connaissances avancées des technologies web ont permis d'aboutir à la réalisation de ce mariage exotique.

Next

Un jeu de hacking pour apprendre le web sémantique

Notre application se présente sous la forme d'un jeu ludique où le joueur interagi avec son environnement via le hacking d'une base de données sémantique.

DreamHacker est un jeu de plateforme connecté à une base de données sémantique où l'interaction se fait grâce à une interface web graphique également reliée à cette base de données.

Technologies

Voici les différentes technologies opérantes dans notre projet


  • Unity 5, Un moteur de jeu multiplateforme
  • Base de données RDF, modélisation sous forme de graphe
  • Jeu multi-joueurs possible

  • Application Web, back end en PHP
  • Application Web, front end avec la librairie GoJs
  • Utilisation Cross Platform possible

Verdict

Malgré notre retard au lancement dû à notre équipe formée sur place, nous avons rapidement réussis à nous fédérer autour d'un projet qui a rassemblé toutes nos envies.
Nous avons évolué rapidement en nous fixant des objectifs très simples au début, afin d'avoir toujours une version opérationnelle, puis nous avons rajouté des fonctionnalités et des éléments graphiques plus complexes.
Et c'est ainsi que nous avons décroché la première place !
